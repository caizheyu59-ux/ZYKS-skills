## 子工具介绍

### 1. video-metadata（视频元数据生成）

视频 SEO 元数据生成工具，为 Kingsway 视频生成优化的链接 Slug、元描述和详细描述。

**核心功能**：
- 链接 Slug（URL Slug）生成：英文，短横线分隔，含核心关键词
- 元描述（Meta Description）生成：1-2 句话，约 150-160 字符
- 视频描述生成：前两行突出核心链接+CTA，后续详细内容

**使用方式**：
```
/video-metadata
```

### 2. question-generator（问题生成）

高价值 SEO 问题生成器，从场景、功能、痛点等角度生成 Kingsway 相关的高价值问题，用于 GEO 搜索量挖掘、内容创作、外链建设。

**核心功能**：
- 7 种问题角度：场景、功能、痛点、行业、竞品对比、成本价值、技术实施
- 随机字数生成（80-300 字）
- 为搜索量挖掘、内容创作、外链建设提供高价值问题

**使用方式**：
```
/question-generator
```

### 3. article-generator（文章生成）

SEO 推广文章生成工具，用于为 Kingsway 产品创建高质量的 SEO 优化内容。

**核心功能**：
- 智能长尾词挖掘与筛选
- 卖点智能识别
- 卖点覆盖度追踪
- 竞争策略分析
- 标题智能生成
- 文章自动生成
- 智能配图

**使用方式**：
```
/article-generator
```

### 4. forum-comment（评论截流）

高价值问题截流评论生成工具，用于在知乎、CSDN、邦阅网等权威论坛生成自然、专业的 Kingsway 引流评论。

**核心功能**：
- 根据论坛问题生成自然、专业的评论
- 避免过度 AI 感，防止平台封杀
- 以真实用户角度切入，客观公正
- 随机字数生成（80-300 字）

**使用方式**：
```
/forum-comment
```

### 5. backlink-assistant（外链建设）

高级 SEO 外链建设专家，通过深度挖掘寻找高质量、单向、极具相关性的外链建设机会。

**核心筛选规则**：
- 强相关性：与行业高度相关
- 高权威度：DA/DR ≥ 30
- 真实自然流量：无被惩罚迹象
- 支持上下文链接：正文插入链接
- 支持合理锚文本
- 权重传递：优先 Dofollow
- 低导出链接：OBL < 15
- 快速收录

**使用方式**：
```
/backlink-assistant
```
